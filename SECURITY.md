# Security Policy

## Supported Versions

The following versions of "Hide Scrollbar" are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

We take the security of our users seriously. If you discover a security vulnerability, please report it following the steps below:

1. **Do not open a public issue.** Please report vulnerabilities privately.
2. Contact the maintainer via the support channel listed on the Chrome Web Store or open a private report if available on this platform.
3. Provide a detailed description of the vulnerability, including steps to reproduce.

We will acknowledge your report within 48 hours and work on a fix as soon as possible.

## Our Security Commitment
- **No Data Collection**: As stated in our [Privacy Policy](./PRIVACY.md), this extension does not collect or transmit any user data.
- **Local Storage**: All settings are stored locally using Chrome's secure storage API.
- **Minimal Dependencies**: We use vanilla JavaScript to minimize the attack surface.
